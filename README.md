# 📊 Student Grade Analyzer

A Python-based data analysis tool that processes student exam scores, calculates performance metrics, assigns grades, and generates visualizations.

## 🎯 Features

- **Data Processing**: Reads student scores from CSV files
- **Score Calculations**: Computes total scores and averages across subjects
- **Grading System**: 
  - Letter grades (A, B, C, D, Failed)
  - Grade points (5-point scale)
  - Performance remarks
- **Ranking**: Automatically ranks students by performance
- **Data Visualization**: 
  - Bar chart for individual performance
  - Scatter plot for score distribution
  - Distribution plot with trend line (KDE)
